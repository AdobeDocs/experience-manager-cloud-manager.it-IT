# Note sulla versione per Cloud Manager 2022.5.0 {#release-notes}

In questa pagina sono documentate le note sulla versione di [!UICONTROL Cloud Manager] 2022.5.0.

>[!NOTE]
>
>Per le ultime note sulla versione di Cloud Manager in AEM as a Cloud Service, vedi [Cloud Manager nelle note sulla versione corrente di AEM as a Cloud Service](https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/implementing/using-cloud-manager/release-notes-cloud-manager/release-notes-cm-current.html?lang=it).

## Data di rilascio {#release-date}

La data di rilascio per [!UICONTROL Cloud Manager] 2022.5.0 è il 5 maggio 2022. La prossima versione è pianificata per il 9 giugno 2022.

## Novità {#what-is-new}

* L’esecuzione della pipeline ora presenta nella schermata di esecuzione gli errori di livello superiore dell’interfaccia utente.
* La configurazione predefinita è stata aggiornata per includere la pausa nella configurazione in background.
* È possibile eseguire nuovamente il passaggio di distribuzione in produzione tramite l’interfaccia utente di Cloud Manager.

## Correzioni di bug {#bug-fixes}

* Risultato del test delle prestazioni fuorviante con errore di timeout 504.
* Alcune esecuzioni rimanevano bloccate nel contesto del trigger di conferma e dell’annullamento del passaggio di approvazione.
